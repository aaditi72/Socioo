# Socioo

**Socioo** is a simple social media web application built using **PHP**, **MySQL**, and **XAMPP**. It allows users to register, log in, create posts, like, comment, follow others, and join communities â€” similar to Reddit.

---

## Features

- User registration and login system  
- Create, edit, and delete posts  
- Like and comment on posts  
- Follow/unfollow users  
- Create and join communities  
- Secure session management

---

## Tech Stack

- **Frontend**: HTML, CSS  
- **Backend**: PHP  
- **Database**: MySQL  
- **Server**: XAMPP (Apache, MySQL)
